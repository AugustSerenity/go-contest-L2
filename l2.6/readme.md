Что выведет программа?

Объяснить поведение срезов при передаче их в функцию.
```go
package main

import (
  "fmt"
)

func main() {
  var s = []string{"1", "2", "3"}
  // Исходный срез указывает на массив: s [1, 2, 3]
  modifySlice(s) 
  // Исходный срез указывает на массив: s [3, 2, 3] 
  fmt.Println(s)
  // Вывод [3, 2, 3] 
  // Потому что со срезом s, который указывал на исходный массив в функции modifySlice
  // работали до первого append, после этого срез i стал указывать на новый массив, 
  // а s остался с прежним
}

func modifySlice(i []string) {
  i[0] = "3" // Изменение в исходном массиве: s теперь [3, 2, 3]

  i = append(i, "4") // Создается новый массив т.к. cap == len
                     // Все элементы копируются в новый массив: [3, 2, 3, 4]
                     // i теперь указывает на этот новый массив, 
                     // исходный срез s продолжает указывать на старый массив [3, 2, 3]

  i[1] = "5" // Меняет только новый массив: [3, 5, 3, 4]
  i = append(i, "6") // Новый массив становится [3, 5, 3, 4, 6]
}
```
