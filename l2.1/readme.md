Что выведет программа?

Объясните вывод программы.

```go
package main

import "fmt"

func main() {
  a := [5]int{76, 77, 78, 79, 80}
  var b []int = a[1:4]
  fmt.Println(b)
}
```

Ответ:
```
[77, 78, 79]
```
массив `b` включает в себя элементы массива `a` с индексами от 1 включительно до 4 не включительно
